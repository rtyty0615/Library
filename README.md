# Library
Book Library Tracker
- A dynamic web app to manage your reading list with add/delete/toggle-read features.

Features
✅ Add books via centered modal dialog
✅ Toggle read/unread status with CSS animation  
✅ Delete individual books
✅ Clean card layout (title/author/pages/status)
✅ Form validation/reset on cancel
✅ Event delegation for dynamic buttons

Tech Stack
HTML5 | CSS3 (Flexbox/Grid) | Vanilla JavaScript (ES6+)
- Constructor functions & prototypes
- DOM manipulation (createElement/appendChild)
- Event delegation (container.addEventListener)
- Crypto.randomUUID() for unique IDs
- Native <dialog> element

Key Learning Outcomes
• Dynamic UI updates without page refresh
• Form handling & data persistence in arrays  
• CSS centering techniques (flex/transform)
• Event bubbling for dynamic elements
• Clean separation: data (myLibrary) ↔ display (addBookToPage)
